# HTML Basics

This repository provides a simple guide to basic HTML tags, attributes, headings, paragraphs, and inline CSS.

---

## Key Topics:

### 1. Basic HTML Tags:
- `<html>`, `<head>`, `<body>`, `<h1>`, `<p>`, `<a>`,<ul>,<ol>,<input>,<label> etc.

### 2. Attributes:
- Tags can have attributes like `href` (for links) or `src` (for images).
  
### 3. Headings:
- `<h1>` to `<h6>` define headings, from largest (`<h1>`) to smallest (`<h6>`).

### 4. Paragraphs:
- Use `<p>` to define paragraphs of text.

### 5. Inline CSS:
- Style elements directly with the `style` attribute, e.g., `<p style="color: blue;">`.
